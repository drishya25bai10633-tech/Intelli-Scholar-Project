# Problem Statement & Scope

## 1. Problem Statement
Engineering students often face "information overload" when trying to align their current academic performance with complex career goals in specialized fields like AI, Machine Learning, or Tech-Art. There is a lack of integrated, intelligent systems that can simultaneously predict academic outcomes based on current habits and provide automated, logic-based reasoning for course eligibility based on university prerequisites.

## 2. Scope of the Project
This project is a **Hybrid Intelligent Agent System** that:
* **Predicts Performance:** Employs a Supervised Learning (Linear Regression) model to forecast GPA based on study patterns and attendance.
* **Reasons Eligibility:** Utilizes a Prolog-based Knowledge Representation engine to verify if a student meets the "First Order Predicate Logic" requirements for advanced modules.
* **Maps Career Paths:** Acts as a Goal-Based Agent to suggest specific academic "Search Strategies" for reaching targets like VFX Artistry or Data Science.

## 3. Target Users
* **Engineering Students:** Specifically those at VIT Bhopal seeking clarity on their AI/ML specialization roadmap.
* **Academic Advisors:** Seeking a tool to automate the checking of complex course prerequisite "Family Trees".
* **Self-Learners:** Individuals looking to understand the practical bridge between Statistical Decision Theory and Logic Programming.

## 4. High-Level Features
* **ML Predictive Engine:** A quantitative module that calculates predicted success metrics using Scikit-learn.
* **Prolog Logic Base:** A robust rule-based system handling facts, predicates, and variables to ensure academic integrity and The system utilizes a 10-point scale for performance prediction, consistent with the grading standards of VIT Bhopal.
* **Informed Search Module:** A pathfinding component that identifies the most efficient sequence of elective subjects.
* **Automated Reporting:** Generates categorized feedback on career readiness and enrollment eligibility.
